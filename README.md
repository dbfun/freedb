Проверка баз данных MySQL на доступность
========================================

Проверяются базы данных MySQL на доступность. Данные хранятся и обновляются в таблице `dbs`.

Установка
---------

    cp config/db.distr config/db
    vim config/db
    mysql database < data/db.sql
  
Проверка
--------

    ./check.sh
